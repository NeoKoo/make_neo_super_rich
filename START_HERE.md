## 💡 使用技巧

### 提高选号效率
1. **使用随机策略**：不要每次都手动选，试试不同的随机策略
2. **关注冷热号码**：在历史记录中查看哪些号码出现频率高
3. **记录策略效果**：保存时记录使用的策略，对比中奖效果

### 利用PWA优势
1. **安装到桌面**：像原生应用一样使用，更方便
2. **离线使用**：网络不佳时仍可查看历史记录
3. **添加到主屏幕**（移动端）：一键打开应用

### 个性化建议
1. **准确填写生日**：幸运色计算依赖准确日期
2. **尝试不同策略**：不同日期可能有不同幸运色
3. **定期清理记录**：保持历史记录整洁，避免存储溢出

## 🐛 常见问题

### Q: 应用启动失败怎么办？

**A**:
1. 检查Node.js版本：`node -v`（需要>=18.x）
2. 删除依赖重新安装：
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```
3. 检查端口占用：确保3000端口未被占用

### Q: API请求失败怎么办？

**A**:
1. 检查.env文件是否配置了API_KEY
2. 访问聚合数据控制台查看额度
3. 暂时可以不配置API，应用仍可使用基础功能

### Q: 样式显示不正确怎么办？

**A**:
1. 清除浏览器缓存：Ctrl+Shift+Delete
2. 强制刷新：Ctrl+F5
3. 检查TailwindCSS是否正确加载

### Q: PWA无法安装怎么办？

**A**:
1. 确保使用HTTPS访问（Vercel自动提供）
2. 检查manifest.json配置
3. 在浏览器开发者工具中查看错误
