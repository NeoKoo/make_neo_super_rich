# 图标说明

本目录存放PWA应用图标，需要以下尺寸的PNG图片：

## 要求的图标尺寸

- icon-72x72.png (72x72)
- icon-96x96.png (96x96)
- icon-128x128.png (128x128)
- icon-144x144.png (144x144)
- icon-152x152.png (152x152)
- icon-192x192.png (192x192)
- icon-384x384.png (384x384)
- icon-512x512.png (512x512)

## 图标设计建议

**主题**：发财、财富、幸运色
**颜色**：紫色系、金色、木质棕
**元素**：
- 金币/元宝
- 招财猫
- 财神
- 财富符号
- 幸运星

## 临时方案

当前使用SVG格式的favicon.ico作为占位符。建议使用在线工具生成专业图标：

1. **Favicon Generator**
   https://realfavicongenerator.net/

2. **Canva**
   https://www.canva.com/

3. **Icons8**
   https://icons8.com/

4. **AI生成**
   使用DALL-E、Midjourney等AI工具生成"发财主题紫色彩票图标"

## 替换步骤

1. 生成上述尺寸的PNG图标
2. 将文件重命名为对应名称
3. 替换本目录中的文件
4. 清除浏览器缓存
5. 重新安装PWA查看效果
