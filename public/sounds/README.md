# 音效文件说明

这个目录包含应用的音效文件。所有音效文件都应该是 MP3 格式。

## 需要的音效文件

以下音效文件需要添加到此目录：

1. `number_select.mp3` - 选号音效（当用户选择号码时播放）
2. `number_clear.mp3` - 清除音效（当用户取消选择号码时播放）
3. `win_celebration.mp3` - 中奖庆祝音效（当用户中奖时播放）
4. `scratch_card.mp3` - 刮刮乐音效（当用户刮开刮刮乐时播放）
5. `button_click.mp3` - 按钮点击音效（通用按钮点击音效）
6. `ai_recommendation.mp3` - AI推荐音效（当AI生成推荐时播放）
7. `save_success.mp3` - 保存成功音效（当保存选号成功时播放）
8. `strategy_select.mp3` - 策略选择音效（当选择随机策略时播放）
9. `tab_switch.mp3` - 标签切换音效（当切换底部标签时播放）
10. `notification.mp3` - 通知音效（当显示通知时播放）

## 音效文件要求

- 格式：MP3
- 采样率：44.1kHz
- 比特率：128kbps
- 时长：建议 0.1-2 秒
- 音量：标准化到 -3dB

## 推荐的音效特征

### 选号音效
- 清脆、积极的音效
- 时长：0.2-0.5秒
- 音调：中高音调

### 清除音效
- 柔和的取消音效
- 时长：0.2-0.4秒
- 音调：中低音调

### 中奖庆祝音效
- 欢快、庆祝的音效
- 时长：1-2秒
- 可以包含多个音符

### 按钮点击音效
- 简洁的点击音效
- 时长：0.1-0.2秒
- 音调：中音调

## 获取音效资源

您可以从以下资源获取免费音效：

1. **Freesound.org** - 免费音效库
2. **Zapsplat** - 免费音效（需要注册）
3. **Adobe Stock Audio** - 专业音效资源
4. **自创音效** - 使用音频编辑软件创建

## 注意事项

- 确保音效文件有合适的版权许可
- 音效文件名必须与上述列表完全匹配
- 所有音效文件都应该小写
- 建议音效文件大小控制在 50KB 以内
